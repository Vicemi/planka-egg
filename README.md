## 📦 Descripción

Este repositorio contiene un **Egg para Pterodactyl Panel** que permite instalar y ejecutar fácilmente **Planka**, una plataforma de gestión de proyectos tipo **Kanban**, completamente autohospedada.

Con este egg podrás desplegar Planka dentro de tu infraestructura de forma rápida, organizada y compatible con el ecosistema de Pterodactyl.

---

## 🚀 Características

- Instalación simplificada en **Pterodactyl**
- Compatible con entornos Docker
- Configuración rápida
- Despliegue limpio y mantenible
- Ideal para servidores autogestionados

---

## 🛠 Requisitos

Antes de usar este egg necesitas:

- Panel **Pterodactyl**
- Node configurado
- Docker operativo
- Acceso administrativo al panel

---

## 📥 Instalación

Importa el archivo JSON del egg en tu panel de administración de Pterodactyl:

`Admin -> Nests -> Import Egg`

Luego crea un nuevo servidor usando el egg de **Planka**.
